方針:
- Chapter1 は `Caliber` / キャリバー・O 固定。チャプター画像 `gate_000` も固定。
- Chapter20 は `Arcana` / アルカナ。チャプター画像 `gate_019` が終焉ゲート。

- 実装IDは Claude 側で Entity Database 登録名に合わせて最終確定する。素材名準拠なら `neutral_*` 接頭辞を使う。
