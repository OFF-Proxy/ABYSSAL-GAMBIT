# 13 Mechaz0r Emotional Map

## 目的

Chapter13〜18のメカゾ0ア連章を、単なる機械ボス6連戦にしないための感情設計です。  
各章に「最も刺さる主人公」を置き、その主人公の答えを終盤で試します。

メカゾ0アはアルカナの終幕思想が機械的に独り歩きした姿です。  
アルカナ本人よりも冷たく、迷いがなく、救済ではなく処理として終わりを実行します。

| 章 | ボス | テーマ | 担当主人公 | 試される答え |
| --- | --- | --- | --- | --- |

| 18 | メカゾ0ア・砲 | 殲滅 | ブローム | 世界ごと撃ち抜く答えに、違う者同士の結束で抗う。 |

## Chapter13: 翼 / レヴァ

### 感情の核

翼は自由の象徴です。  
メカゾ0ア・翼は、飛ぶ自由を「墜落の危険」として処理します。

レヴァはどこへでも行ける自由を持つキャラクターです。  
だからこそ、この章では「逃げる自由」ではなく「残る自由」を選ばせます。

### 戦前台詞

```text
メカゾ0ア・翼: ORDER: 飛行角度を固定。逸脱する自由を停止せよ。
レヴァ: 自由って、逃げるためだけのものじゃないよ。ここに残るのも、私が選んだ飛び方だ。
メカゾ0ア・翼: RESPONSE LOGGED。墜落可能性、戦闘で検証する。
```

### 撃破後

```text
メカゾ0ア・翼: 固定不能。飛行経路、未確定。
レヴァ: 未確定でいいじゃん。空って、そういう場所でしょ。
```

## Chapter14: 剣 / アルディン

### 感情の核

剣は、迷いと後悔を切れば楽になると提示します。  
アルディンは守れなかった後悔を抱えています。  
この章では、後悔を切り捨てるのではなく、守る理由として抱え続ける答えを出します。

### 戦前台詞

```text
メカゾ0ア・剣: ORDER: 後悔を切断。守護効率を最適化する。
アルディン: 後悔を切れば、守れなかった名まで落ちる。私はその重さごと剣を取る。
メカゾ0ア・剣: 重量過多。切断を実行する。
```

### 撃破後

```text
メカゾ0ア・剣: 切断失敗。対象、後悔を保持。
アルディン: 保持ではない。背負って進むと言うんだ。
```

## Chapter15: 完全体 / イレーナ

### 感情の核

完全体は「もう欠けない世界」を提示します。  
しかし欠けない世界には、選択も余白もありません。

イレーナは観測者に最も近い主人公です。  
この章で彼女は、完全な結論を拒みます。

### 戦前台詞

```text
メカゾ0ア完全体: COMPLETE。欠落、誤差、祈り、すべて統合済み。
イレーナ: 完成した記録に、未来は追記できません。私は未完成を選びます。
メカゾ0ア完全体: 未完成は不具合。修正を開始する。
```

### 撃破後

```text
メカゾ0ア完全体: 完成状態、崩壊。
イレーナ: 余白が戻っただけです。そこから、世界は続きます。
```

## Chapter16: 兜 / シダイ

### 感情の核

兜は「考えなくていい、命令に従えば楽だ」と語ります。  
シダイは影として誰かを支える人物ですが、命令で動く無名の部品ではありません。

この章では、影にも名があることを明確にします。

### 戦前台詞

```text
メカゾ0ア・兜: ORDER: 自己判断を停止。命令系統へ接続せよ。
シダイ: 俺は誰かの影だ。だが、命令で動く空の兜じゃない。
メカゾ0ア・兜: 名称不要。役割のみ保持する。
```

### 撃破後

```text
メカゾ0ア・兜: 役割分解。名称、残存。
シダイ: 影にも名がある。覚えておけ。
```

## Chapter17: 器 / ヴェスナ

### 感情の核

器は、全員を同じ型へ揃えれば争いが減ると提示します。  
ヴェスナの答えは、居場所は同一化ではなく、違う温度を置ける場所だというものです。

### 戦前台詞

```text
メカゾ0ア・器: ORDER: 個別名を削除。同一規格へ成形する。
ヴェスナ: 居場所は、同じ形に削ることじゃない。違う火が隣にあっても凍えない場所だ。
メカゾ0ア・器: 熱源差異を検出。均一化する。
```

### 撃破後

```text
メカゾ0ア・器: 規格統一、失敗。
ヴェスナ: 形が違うから、誰かの隣に置ける火もある。
```

## Chapter18: 砲 / ブローム

### 感情の核

砲は、苦しみを生む世界ごと撃ち抜く答えです。  
ブロームは、違う者同士でも同じ明日を支えられると信じています。

この章はメカゾ0ア連章の終点です。  
個別の自由、後悔、未完成、名、居場所を守ってきた答えが、最後に「結束」へ集まります。

### 戦前台詞

```text
メカゾ0ア・砲: TARGET: MITHRA。苦痛発生源を世界単位で消去する。
ブローム: 世界は標的ではない。違う者が、まだ同じ明日を支えている。
メカゾ0ア・砲: 支持構造ごと消去する。
```

### 撃破後

```text
メカゾ0ア・砲: 照準、喪失。世界、未消去。
ブローム: 未消去ではない。守られた、と記録しろ。
```

## 実装メモ

- 担当主人公以外でも章は成立する。
- 担当主人公で挑んだ場合だけ、専用3行または撃破後2行を優先する。
- 実装キーは `bossId|heroId`。
- 担当主人公以外は既存の章別汎用返答でよい。
- Chapter13〜18で `INT_13` を再生してはいけない。最終説得はChapter20。

## Chapter20への接続

Chapter20でアルカナは、メカゾ0ア連章を「自分の終幕思想が、自分の手を離れた証」として認識します。

```text
アルカナ: あれは、私が望んだ静けさの影でした。
イレーナ: 影に意志を渡せば、観測者さえ置き去りにされます。
アルカナ: ええ。終わりすら、私の手には残りませんでした。
```
