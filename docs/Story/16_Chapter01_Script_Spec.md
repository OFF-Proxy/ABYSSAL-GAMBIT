# 16 Chapter01 Script Spec

Chapter01「白門の後悔」の実装台本を作るための仕様書です。  
他セッションで台本を書く場合は、本書と `CODEX_PROMPT_Chapter01_FullScript.md` を出発点にします。

## 目的

Chapter01は、プレイヤーが最初に本作の魅力へ気付く入口です。  
ここで伝えるべきことは、世界観の説明ではなく、次の四つです。

- この世界は「終幕の眠り」によって静かに壊れ始めている。
- 主人公は「誰の答えで世界へ挑むか」を選ぶ存在である。
- ボスは倒すだけの敵ではなく、その思想ごとこちら側の力になる。
- キャリバーはChapter01でユニット加入するが、完全に心を許したわけではない。

## 最新採用方針

| 項目 | 採用内容 |
| --- | --- |
| Chapter01ボス | キャリバー / `Caliber` |
| 初回クリア報酬 | `Caliber` の戦術ユニット仮加入、剣片契約、観測片解放 |
| 再クリア報酬 | キャリバー強化。Chapter01周回に必ず恩恵を持たせる |
| 長期回収 | Chapter14でアルディン覚醒と連動し、正式共闘 / 誓剣解放 |
| アルカナ名 | Chapter01では出さない。「白い声」「優しい声」「誰かに観られていた感覚」まで |
| キャリバーの結論 | 改心ではない。主人公の道を見届けるため剣片を預ける |

## Chapter01の役割

Chapter01はチュートリアル章でありながら、IPの入口です。  
単に「世界が危ない」と説明する章ではなく、プレイヤーに以下の感情を順番に持たせます。

1. 静かで異常な世界に入った。
2. アルディン、カガチ、ヴェスナの三人が気になる。
3. キャリバーは悪人ではなく、後悔で壊れた英雄だ。
4. 倒したボスがすぐ使える。
5. ただし、この因縁はまだ終わっていない。

## 登場人物

### キャリバー

- 表示名: キャリバー。UIではキャリバー・Oでも可。
- Unit ID: `Caliber`
- 一人称: 私
- 口調: 厳粛、硬い、騎士的。怒鳴らず、刃のように重く言う。
- 核: 守れなかった痛みを、終わらせる慈悲へ変えてしまった英雄。
- アルディンとの関係: 兄弟弟子。白門とリオラの因縁を共有する。
- 禁止: 完全な悪役化、軽口、即改心、洗脳だけで説明すること。

### アルディン

- Unit ID: `HeroAldin`
- Chapter01の最重要主人公。
- キャリバーの兄弟弟子。
- 白門とリオラを守れなかった後悔を抱える。
- 答え: 守れなくても守り続ける。

### カガチ

- Unit ID: `HeroKagachi`
- 口が悪いが、誰より先に前へ出る。
- Chapter01では「弱くても進む」性格を早く見せる。
- Chapter08犬化と覚醒への遠い前振りとして、転ぶ、強がる、すぐ立つ描写を入れてよい。

### ヴェスナ

- Unit ID: `HeroVesna`
- 凍えた場所に火を置く人。
- Chapter01では避難者や沈黙した広場に「居場所は作れる」を見せる。
- 火の比喩は使ってよいが、攻撃的な炎ではなく温度として扱う。

## 初回プレイとリプレイ

初回プレイの導入は、初期主人公三人を軸に作ります。

- `HeroAldin`
- `HeroKagachi`
- `HeroVesna`

ただしChapter01は後から再挑戦されるため、章ボス戦前と撃破後の台詞は18主人公分を用意します。

## 必須出力

台本作成時は、以下のセクションを必ず作ります。

| セクション | ID案 | 必須内容 |
| --- | --- | --- |
| 章開始幕間 | `CH01_OPENING` | 終幕の眠り、白門跡、初期三人の魅力、キャリバー暴走の提示 |
| 最初の戦闘前 | `CH01_NODE_01` | 最初の門へ入る短い会話 |
| キャリバー遠隔台詞 | `CH01_CALIBER_ECHO_01` など | 章中に3回。キャリバーの思想が徐々に分かる |
| 中ボス台詞 | `CH01_MID_01` など | 4〜6体分。キャリバーの思想を予告する前座 |
| 章ボス戦前 | `CH01_PREFIGHT_<heroId>` | 18主人公分。原則3行 |
| 章ボス撃破後 | `CH01_POSTBOSS_<heroId>` | 18主人公分。原則3行から5行 |

## 台詞量

| 種類 | 目安 |
| --- | --- |
| 章開始幕間 | 20〜35行 |
| キャリバー遠隔台詞 | 各1〜2行 |
| 中ボス台詞 | 各3行 |
| 章ボス戦前 | 各3行 |
| 章ボス撃破後 | 各3〜5行 |
| 報酬演出 | 3〜6行 |

1行は長くしすぎない。  
ゲーム内表示を想定し、長い文章は二行に割ります。

## 章開始幕間の必須ビート

章開始幕間は、説明より体験を優先します。

必須ビート:

1. 鐘が鳴らない白門跡。
2. 眠っているが死んでいない人々。
3. アルディンが静かに避難者を守る。
4. カガチが先に飛び出し、強がる。
5. ヴェスナが凍えた場所に火を置く。
6. キャリバーが遠くから「白門を閉じろ」と叫ぶ。
7. 誰かの白い声、優しい声の気配だけが残る。

## キャリバー加入演出

Chapter01初回クリア時、キャリバーはユニットとして使えるようになります。  
ただし物語上は「正式な仲間」ではなく「剣片契約」です。

必須ニュアンス:

- キャリバーは主人公の答えを完全には認めない。
- だが、その道が折れるか見届けるため、剣片を預ける。
- プレイヤーはすぐに `Caliber` を使える。
- Chapter01再クリアでキャリバーは強化される。
- Chapter14で正式共闘 / 誓剣解放へ発展する。

推奨台詞:

```text
キャリバー: お前の守護を認めたわけではない。
キャリバー: だが、その道が折れるかどうかは見届ける。
SYSTEM: キャリバーが戦術ユニットとして仮加入しました。
SYSTEM: キャリバーの剣片を獲得。
```

## 観測片

観測片は既存正本を使います。  
大きく改稿しないでください。

```text
タイトル: 白門の慈悲

終焉は慈悲だと、誰かが言った。
倒れた子の名を呼ぶ声も、瓦礫の下で途切れた祈りも、もう増えなくて済むから。
けれど私は知っている。
増えない痛みは、癒える時間も持たない。
鐘は鳴らなかった。
それでも、誰かが鐘楼へ登ろうとしていた。
```

観測片取得後に、アルカナの名前は出しません。  
「誰かが見ていた」「白いページが一瞬だけ開く」程度に留めます。

## 中ボス方針

中ボスはキャリバーより目立たない。  
役割は「キャリバーの後悔が周囲へ伝播している」と見せることです。

候補:

- 白門の残兵
- 眠りに寄った騎士
- 門前の獣
- 鐘楼の監視者
- 白い札を持つ祈り手
- 鎖を守る岩顎

中ボスは名前を持ってよいですが、濃すぎる背景は不要です。

## Chapter14への伏線

Chapter01では、Chapter14の結論を言い切りません。

伏線として入れてよい要素:

- キャリバーの剣にヒビが入る。
- アルディンが「後悔を切り捨てない」と言う。
- キャリバーが「重すぎる」と返す。
- 剣片がプレイヤー側へ残る。

言ってはいけないこと:

- Chapter14で正式共闘するという説明。
- 誓剣キャリバーの名前をChapter01内で明かすこと。
- アルディンが覚醒することの直接予告。

## 禁止事項

- Chapter01でアルカナの名前を出す。
- キャリバーを完全な悪人として書く。
- キャリバーを完全に改心させる。
- キャリバーをChapter14まで使用不可にする。
- 主人公の台詞を説教にする。
- 初回から設定用語を並べすぎる。
- アルディン、カガチ、ヴェスナの三人を説明役だけにする。

## 品質チェック

- 最初の5分で異常な世界と初期三人の魅力が伝わるか。
- キャリバーの後悔が、悪ではなく歪んだ守護として見えるか。
- ボス撃破後、すぐユニットとして使える喜びがあるか。
- それでもChapter14へ続く未解決感が残っているか。
- アルカナの伏線が名前なしで匂っているか。
- Chapter02のオルムへ自然につながるか。
