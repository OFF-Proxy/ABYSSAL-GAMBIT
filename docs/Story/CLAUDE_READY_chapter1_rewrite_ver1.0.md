# Chapter1 改稿稿: 英雄Caliber暴走版

[prologue chapter=1]
1: 終幕の眠りは、音も刃も立てずに世界の端から広がり、人々の朝を静かに奪い始めていた。
2: 人は死なない。ただ痛みも願いも選ぶ力も抜かれ、穏やかな顔で戻らぬ眠りへ落ちていく。
3: 各地に二十の門が現れ、強者と災厄が門守りとなって、目覚めようとする者の道を塞いだ。
4: 門の奥には、終焉を慈愛のように観測する者、アルカナが待つという噂だけが静かに残った。
5: 眠りを解く術を求めるなら、門を越え、観測片を集め、その名のもとへ辿り着くしかない。
6: 最初の白門跡には、炎と千切れた鎖の中で、翠の片眼を燃やす英雄キャリバーが立っていた。

N: 彼は守護の象徴だった。白門の陥落を生き延び、誰より傷を抱えながら、それでも巨剣を捨てなかった英雄だった。

N: 空には門が開いている。白く、静かで、祈りにも墓標にも見えるそれが、世界の終わりをひとつずつ数えていた。
主人公: 眠りを解く術を探す。門を越え、観測片を集め、アルカナへ辿り着く。誰かが終わりを決める前に、まだ選べると示すために。
N: 止めるために向かう者もいる。偶然門前で襲われ、刃を抜く者もいる。理由は違っても、最初の門は同じだった。
主人公: 英雄を止める。襲ってくるなら、なおさら目を覚まさせなければならない。守護者の名を、眠りの命令で終わらせはしない。
