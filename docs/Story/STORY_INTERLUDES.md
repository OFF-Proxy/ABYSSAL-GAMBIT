## INT_01 — 第一合流（発生: ch4〜6 / マグマー編）
- 物語機能: 各個撃破では原初の猛威に抗えず、**ブローム主導で守護三陣営が初めて一つの軍になる**。
- 登場: 主人公＋ブローム＋（合流先の別陣営ヒーロー1〜2）。主人公が誰かで合流相手を出し分け。
- 種/トーン: 結束。違う思想・因縁を束ねる難しさ→「違う者同士でも背中は預けられる」。
- 必要演出/実装: 戦闘外VN枠。複数立ち絵の並列表示。
- 長さ: 18〜30行。
- 本文:
```
[interlude=INT_01]
（ト書き: マグマーの熱で地平が赤く揺れる。別々に進んでいた隊が、崩れた石橋の前で足を止める）
ブローム: ここから先は、ひとつの旗で進む。
カガチ: 旗？ 俺は誰かの後ろを歩く気はねえ。
レヴァ: じゃあ横でいいじゃん。後ろより狙いやすいし。
ヴェスナ: 近づきすぎると、私の炎が移る。
カーラ: 移っても消せる。離れすぎる方が危ない。
アルディン: 守る位置が増えるなら、私は歓迎する。
ブローム: 守る者、進む者、火を抱く者。役目は違う。
カガチ: 違うから足を引っ張ることもある。
ブローム: その時は、引っ張られた側が踏み直せ。
レヴァ: いいね。雑だけど嫌いじゃない。
ヴェスナ: それで、本当に信じられるのか。

アルディン: まずは一度、生きて橋を渡る。それで十分だ。
ブローム: マグマーは群れではない。熱と本能の奔流だ。
カガチ: なら、こっちは意思で流れを切る。
ヴェスナ: 焼き尽くす火には、温める火で抗う。
レヴァ: 空は私が見る。変なのが飛んだら落とす。
カーラ: 盾は前へ。退路も残す。
ブローム: よし。違うままでいい。
ブローム: 違うまま、一つになれ。
（ト書き: それぞれの陣営章が重なる。熱風の中で、初めて同じ足音が鳴る）

## INT_08 — カガチ犬化（発生: ch8 / アビサル・キャシヴァ｜カガチ専用｜断片: 生は苦しみだけではない）
- 物語機能（確定 v1）: キャシヴァがカガチの「強さの鎧」を剥ぐために犬化。犬＝屈辱でなく**剥き身の生命の記号**（痛い・腹が減る・匂いが分かる・仲間の温度に反応）。刃も威圧も失った姿で前へ進み、「弱くても進める」が身体で成立する。
- 構成（必須ビート）: ①変身 ②反発/無力 ③シダイ・ジラーンの支え（シダイは影火院を知る同門）④クリア後＝スキン解放の意味づけ（尊厳が戻る。笑いっぱなしで終わらせない）。
- キャシヴァの距離: 完全和解させず「食えない悪友」。刺し方は常に「終わりたいなら終われば？」でなく**「まだ生きるんでしょ？」**。
- 他主人公が章8に来た場合: 変身させず、その主人公の生き汚さ/守りのしつこさ/居場所への熱をからかう（3行は [STORY_DIALOGUE_DRAFTS.md] 汎用を使用）。
- 長さ: 30行以上可（特別枠）。
- 本文:
```
[interlude=INT_08]
（ト書き: キャシヴァの指先から黒紫の光が跳ねる。笑い声は軽く、呪いの重さだけが遅れて落ちる）
キャシヴァ: ふふ。刃も影も、少し重そうね。
カガチ: 何をした。
キャシヴァ: 似合う姿にしてあげただけ。

カガチ: ふざけるな。こんな姿で、どう戦えってんだ。
キャシヴァ: どう生きるか、でしょう？
カガチ: 同じだ。
キャシヴァ: 違うわ。強いふりをしてる時は、よく間違えるの。
（ト書き: カガチが踏み出す。剣を握れない足が滑り、石床に倒れる）
カガチ: くそ……。
キャシヴァ: 痛い？ 悔しい？ お腹も空く？
カガチ: 黙れ。
キャシヴァ: それ、全部生きてる証よ。
シダイ: カガチ。
カガチ: 見るな、シダイ。
シダイ: 影火院で、お前は俺にそう言ったな。
カガチ: ……何の話だ。
シダイ: 「走れ。生きてれば、次は斬れる」って。
カガチ: あれは、お前が弱かったからだ。
シダイ: 違う。お前が俺を背負って逃がしたからだ。
ジラーン: 傷を見せて。姿が変わっても、痛みは痛みです。
カガチ: 俺は、手当てされる側じゃない。
ジラーン: 今日はそうです。明日は違うかもしれない。
キャシヴァ: いいわねえ。刃がなくても、誰かが寄ってくる。
カガチ: 笑ってんじゃねえ。
キャシヴァ: 笑うわよ。転んで、噛みついて、それでも立つんだもの。

## INT_12 — 観測者の正体（発生: ch12前後 / ヴェツルヴィアン｜断片: 終焉の観測者・彼女は世界を愛していた）
- 物語機能: 全ての糸がアルカナへ繋がると判明。**イレーナ**（ヴァナーの記録術でアルカナの断片に触れた学究＝「同じものを見て違う答えを選ぶ者」）が観測者の正体と疲弊を解き明かす。
- 種/トーン: 理解。糾弾でなく「彼女が世界を愛していたこと」への気づき。
- 主人公がイレーナ以外のとき: イレーナは仲間として同席し、解説役を担う（主人公選択式の軸を崩さない）。
- 長さ: 15〜25行。
- 本文:
```
[interlude=INT_12]
（ト書き: 砂の神殿の奥。サイオンの刃が消えたあと、壁面の古い記録が青白く浮かぶ）
イレーナ: これは戦記じゃない。観測記録です。
主人公: 観測記録？
イレーナ: 六勢力が争い、均衡が壊れ、誰かが立て直す。その繰り返し。
レヴァ: ずいぶん嫌な日記だね。
イレーナ: 書いた者は、嫌いで記したわけではありません。
カーラ: なら、どうして終わりへ向かうの。
イレーナ: 愛していたからです。何度も失うには、愛しすぎた。
ブローム: アルカナか。
イレーナ: はい。終焉の観測者。元はミスラの記録官であり、調停者。
アルディン: 守るために、見続けたのか。
イレーナ: そして、見続けるために、同じ時間を生きる権利を手放した。
ヴェスナ: 帰る場所を失った者が、世界ごと眠らせようとしている。
カガチ: ふざけてる。疲れたから全部止めるってのか。
イレーナ: そう言い切れたら楽でした。
イレーナ: 彼女は世界を憎んでいない。だから厄介なんです。
主人公: なら、止めるだけじゃ足りない。
イレーナ: ええ。彼女の観測を否定するのではなく、その先を見せる必要があります。
サイオンの残響: 観測者は、終幕を視た。
イレーナ: それでも、まだ観ていないものがあります。
主人公: 私たちの答えか。
イレーナ: はい。同じものを見て、違う答えを選ぶ。そのために、ここまで来ました。
（ト書き: 記録の光が消える。代わりに、遠くで白い髪の影が一度だけ振り返る）
```

## INT_13 — 最終説得（発生: ch13 / アルカナ撃破後）
- 物語機能（確定 v1）: 撃破で終わらせず、**アルカナの“選択”**で幕。集めた仲間（かつての敵）と理解が、彼女の諦めを覆す。
- 構成: アルカナの「もう終わらせましょう」→ **各主人公の答えを短く束ねる**（全員が長台詞を言わない）→ アルカナが終幕の眠りをやめ、もう一度世界を観ることを選ぶ。
- 演出（確定 v1）: 「終わらせる＝終幕の眠り（静止・色が落ちる・音が消える）」。説得の瞬間に色と音が戻る方向。
- 主人公別: 締めの一言だけ選択中の主人公の“答え”に差し替え（[STORY_ENDINGS.md] と連動）。
- 長さ: 20〜30行＋主人公別差し替え1〜2行。

アルディン: 眠らせれば、守れなかったものも消えるのか。
