# 12 Arcana Private History

## 目的

アルカナを「世界を愛しすぎた観測者」という抽象だけで終わらせないための、個人的な喪失の正本です。  
ここで定義する記憶は増やしすぎない。アルカナの痛みは、三つの具体で十分です。

この三つは、観測片、Chapter20、エンディング、アルカナ仲間化後の台詞で繰り返し使います。

## アルカナの三つの傷

| 種別 | 名前 | 内容 | 物語上の役割 |
| --- | --- | --- | --- |
| 最初に記録した名 | ネリ | 白門より古い辺境の村で、アルカナが初めて名前を記録した子供。村は次の争いで失われ、記録だけが残った。 | アルカナが「名前を残すこと」に希望を持っていた時代。 |
| 最初の友人 | オルハ | アルカナに記録官の役目を教えた先輩。記録は未来への手紙だと語り、調停の失敗で命を落とした。 | アルカナが世界を憎めない理由。 |
| 記録の裏切り | 白紙戦争 | アルカナの記録が、争いを避けるためではなく、次の戦争を効率化する資料として使われた事件。 | アルカナが「記録しても人は変わらない」と折れた決定打。 |

## 傷1: ネリ

ネリは、ミスラの古い辺境にいた子供です。  
アルカナがまだ「記録すれば、次の誰かが同じ悲劇を避けられる」と信じていた頃、彼女はネリの村を訪れました。

ネリは大きな英雄ではありません。  
特別な血筋もありません。  
だからこそ重要です。

彼女はアルカナに、自分の名と、弟の笑い方と、朝に鳴る小さな鐘の音を記録してほしいと頼みます。  
その村は後に、勢力間の小さな衝突で地図から消えます。

アルカナは、世界がネリの名前を忘れても、自分だけは忘れませんでした。  
忘れられないことは、最初は祈りでした。  
長い時間の後、それは呪いになりました。

### 使う場所

- Fragment01: 白門の慈悲
- Chapter1: キャリバーの後悔と重ねる
- Chapter20: アルカナが「名前を残しても、守れませんでした」と漏らす

### 台詞種

```text
アルカナ: 私は、ネリという子の名をまだ覚えています。
アルカナ: 世界は忘れました。村も、鐘も、弟の笑い方も。
アルカナ: 覚えているだけでは、誰も守れませんでした。
```

## 傷2: オルハ

オルハはアルカナに記録官の役目を教えた人物です。  
師匠でもあり、友人でもあり、アルカナが「世界の外へ行く前」に最後まで対等に話せた相手です。

オルハの言葉:

```text
記録は墓標ではない。
まだ会ったことのない誰かへ渡す、未来への手紙だ。
```

アルカナはこの言葉を信じました。  
争いを記録し、調停を記録し、和解の失敗も記録しました。

けれどオルハは、調停が失敗した夜に死にます。  
死ぬ前に、アルカナへ「記録して」と頼みます。  
アルカナは初めて、自分のために泣かず、誰かの最後の願いとして記録します。

この記憶があるため、アルカナは世界を憎めません。  
世界にはオルハのような人がいたからです。

### 使う場所

- Fragment11: 愛したから
- Fragment12: 観測者
- Chapter20: アルカナが「記録は未来への手紙だったはず」と言う

### 台詞種

```text
アルカナ: オルハは、記録は未来への手紙だと言いました。
アルカナ: 私は長いあいだ、その言葉だけで立っていました。
アルカナ: でも手紙は、何度も燃やされました。
```

## 傷3: 白紙戦争

白紙戦争は、アルカナが決定的に折れた事件です。  
彼女の記録は、二度と同じ失敗を繰り返さないために残されたものでした。

しかし、ある王と軍師たちはその記録を別の用途に使います。

- どの門を攻めれば民が逃げられないか。
- どの英雄がどの後悔に弱いか。
- どの祈りを利用すれば、降伏を早められるか。
- どの季節に攻めれば、次世代の報復を抑えられるか。

アルカナの記録は、平和への手紙ではなく、戦争の攻略本になりました。  
この事件の後、彼女は一度、記録をすべて白紙に戻そうとします。  
けれど白紙にしても、死んだ者は戻らない。  
書いても、白紙にしても、世界はまた同じ傷へ向かう。

ここでアルカナは「ならば、痛みも願いも眠らせるしかない」と考え始めます。

### 使う場所

- Fragment10: 誤差
- Fragment18: 照準
- Mechaz0r連章全体
- Chapter20: アルカナが終幕処理を正当化する根拠

### 台詞種

```text
アルカナ: 私の記録は、次の戦争を避けるためにありました。
アルカナ: けれど人は、それで次の戦争を上手に始めました。
アルカナ: ならば記録では足りない。そう思ってしまったのです。
```

## アルカナの変化

| 時期 | アルカナの信念 | 台詞の温度 |
| --- | --- | --- |
| ネリ以前 | 記録すれば、誰かが助かる | まだ人間らしい。迷いも笑いもある。 |
| ネリ後 | 忘れないことは祈りである | 静かだが、希望が残る。 |
| オルハ後 | 記録は未来への手紙である | 丁寧。誰かへ届ける意志がある。 |
| 白紙戦争後 | 記録しても人は変わらない | 丁寧だが、冷え始める。 |
| 終焉の観測者 | 苦しみごと眠らせるしかない | いたわりの口調。怒鳴らない。 |
| Chapter20後 | 観測は諦めの別名ではないかもしれない | 迷いと余白が戻る。 |

## 書く時の注意

- ネリ、オルハ、白紙戦争を説明しすぎない。
- 本編中は、名前を小出しにする。
- Chapter20までは、プレイヤーに「これは誰の記憶か」を完全には言い切らない。
- アルカナが泣き叫ぶ場面は作らない。
- 彼女の痛みは大声ではなく、文末の揺れで見せる。

## Chapter20での使い方

Chapter20では、アルカナが三つの傷を自分から語ります。  
ただし長い独白にしません。戦闘フェーズや観測片共鳴の合間に、短い断片として出します。

推奨順:

1. 戦前: ネリの名を出す。
2. HP 70%: オルハの言葉を出す。
3. HP 40%: 白紙戦争を出す。
4. 撃破後: 三つの記憶を閉じず、余白として残す。

```text
アルカナ: ネリ。オルハ。白紙にした戦争。
アルカナ: どれも、終わらせれば増えない傷でした。
主人公: 増えない傷は、癒える時間も持たない。
アルカナ: ……それは、私が最初に書いた言葉でしたね。
```
