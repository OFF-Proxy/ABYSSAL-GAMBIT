# 10 Ending

## 方針

エンディングは共通です。  
差分は、選択中主人公の最後の答えと一言エピローグに留めます。  
18ルート完全分岐ではなく、18通りの答えが同じ終幕へ別の角度から届く構造です。

## 最終説得の流れ

1. アルカナが「もう十分です」と終幕の眠りを開く。
2. 仲間になったボスと主人公たちが、それぞれの答えを短く返す。
3. 選択中の主人公が最後の答えを言う。
4. アルカナは完全敗北ではなく、初めて迷う。
5. 「続きを……観てみたいですね。」へ至る。
6. 終幕の書が閉じ、世界に音と色が戻る。
7. アルカナが最後の仲間として解放される。

実装投入用のフル台本は `14_Implementation_Dialogue_Scripts.md` の `INT_20_ARCANA_FINAL` を正本とします。  
旧実装の `INT_13` を流用する場合も、再生タイミングはChapter20へ移し、本文はこの正本へ寄せます。

## 共通ラスト台詞

```text
アルカナ: では、もう一度だけ。
アルカナ: あなたたちの続きを、観てみましょう。
```

## アルカナ仲間化

ボスは倒すと仲間になる。  
このゲームのルールは最後まで例外を作りません。

したがってアルカナも、撃破後に仲間化します。  
これは免罪ではありません。彼女が世界の外から降り、プレイヤーの選択の中に入ることです。
