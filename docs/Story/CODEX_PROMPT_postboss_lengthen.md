# Codexへの修正指示：撃破後セリフを丁寧に（長尺化）

> 先に納品された `POSTBOSS_DIALOGUE_CH1_12_CODEX_DRAFT.md`（全組3行）への手直し依頼。
> 下の `===` ブロックを、その草稿と一緒に Codex に貼る。

===========================================================================

先に書いてもらった撃破後セリフ（`[postboss ...]`）は、どの組も3行（ボス→主人公→ボス）の
一往復で、やや淡白でした。**戦いを終えた直後の余韻**として、もう少し丁寧に語らせたいです。
**全組を 5〜7行に増やして**書き直してください（短い余韻＝5行、印象的な組＝7行まで）。

## 行数と構成（厳守）
- **ボスで開始し、ボス→主人公→ボス…と交互**（偶数ラベル=ボス / 奇数ラベル=主人公）。
- 標準は **5行**（ボス・主人公・ボス・主人公・ボス）。重要な因縁のある組は **7行**まで可。
- 既存の良い言い回しは活かしつつ、間に往復を足して「**負けたボスが主人公の答えを受け止めて変化する**」流れにする。

## 撃破後シーンの狙い（ここが戦前と違う）
- これは**勝った後**の会話。ボスはもう敵意を収め、敗北を噛みしめている。罵り合いではなく、
  **主人公の生き様にボスが触れて、少し態度が変わる**余韻にする。
- 1行目：ボスが敗北・主人公の在り方に触れる（皮肉・感嘆・問い）。
- 中盤：主人公が自分の核となる価値観で短く返す → ボスがさらに踏み込んで問う/認める。
- 最終行：**ボスの締め**。完全な改心まで描かず、「次へ送り出す」「わずかな敬意」「未練」程度に留める
  （アルカナへ向かう物語なので、ここで全部を解決しない）。

## 文体（戦前と共通）
- 短く芯のある言い回し。説教くさく・叙情過多にしない。各ボスの思想と各主人公の信条は草稿のまま維持。
- アルカナ系の慈愛口調・キャリバーの兄弟子設定など、固有設定は崩さない。
- カガチ8章キャシヴァ戦は犬化後の文脈（屈辱→生の肯定）を保つ。

## 出力フォーマット（変更なし）
- 見出しキーは**草稿と完全に同じ**にする（`[postboss boss=<bossId> hero=<heroId>]`）。差し替え漏れ防止。
- 対象は草稿と同じ ch1-12 の12ボス × 9主人公＝108組。**全組を出力**してください（増やした分だけ部分提出しない）。

### 5行の例（イメージ。この組を実際に書く必要はない）
```
[postboss boss=Caliber hero=HeroAldin]
1(ボス): アルディン……その盾は、まだ折れぬか。
2(主人公): 折れても掲げる。リオラの名も、白門も、置いてはいけない。
3(ボス): 私はあの日、抱えきれずに刃を選んだ。お前は、抱えたまま立つというのか。
4(主人公): 重さごと進む。それしか、あの子に返せるものがない。
5(ボス): ……ならば進め。私の後悔ごと、その背に乗せてやる。
```
