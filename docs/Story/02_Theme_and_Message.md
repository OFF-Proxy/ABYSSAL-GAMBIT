# 02 Theme and Message

## 中核テーマ

この作品は「悪を倒す物語」ではありません。  
主人公たちは、様々な理由で諦めた者たちと出会い、その思想を理解し、それでも違う答えを選びます。

全主人公に共通する答えは:

> それでも生きる。

ただし、全員が同じ言葉で希望を語ってはいけません。  
アルディンは守護として、カガチは前進として、ヴェスナは居場所として、同じ結論へ別の道から辿り着きます。

## アルカナの問い

アルカナの問いは単純です。

> 何度も失う世界に、それでも続きを信じる価値はありますか？

彼女は世界を憎んでいません。  
愛しすぎたからこそ、これ以上傷つく前に眠らせたい。  
この「理解できてしまう終焉」が、ラスボスとしての彼女の強さです。

## プレイヤーの問い

プレイヤーは章を選び、主人公を選び、仲間になったボスを使います。  
その選択そのものがストーリーです。

- 主人公選択: 誰の答えで挑むか。
- ボス仲間化: どの諦めを理解し、力に変えるか。
- 観測片収集: アルカナの記憶をどこまで引き受けるか。
- 周回: 同じ世界に、違う答えを何度でも置く。

## 書いてはいけないもの

- アルカナを悪役口調にする。
- ボスを倒した瞬間に改心させる。
- 主人公が過去章の情報を当然のように知っている。
- すべての主人公に同じ希望台詞を言わせる。
- 抽象語だけで会話を終える。
- 「世界を救うために倒す」だけで章を終わらせる。

## 書くべきもの

- 敵の思想が一度は正しく聞こえる瞬間。
- 主人公が相手の傷を理解した上で別の答えを出す瞬間。
- 仲間化後もボスの思想が完全には変わらない余韻。
- 観測片が後で意味を持つ伏線。
- UI、BGM、カメラまで含めた物語表現。
