# ABYSSAL GAMBIT Story Bible v2

> **目的**
>
> このドキュメントは、ABYSSAL GAMBIT
> のストーリー・ゲームシステム・キャラクター設計を統一するための最上位仕様書です。
> Codex / Claude /
> 開発者がこのファイルだけを読めば、作品の方向性・実装意図・演出方針を理解できることを目的とします。

------------------------------------------------------------------------

# 1. コンセプト

## ジャンル

2D オートバトラー / ボスラッシュ / ストーリーRPG

## このゲームで一番大切なこと

この作品は

「悪を倒す物語」

ではありません。

主人公達は、

様々な理由で"諦めた者達"と出会い、 その思想を理解し、
それでも違う答えを選ぶ物語です。

ゲームシステムそのものがストーリーになります。

・主人公選択 ・ボス仲間化 ・観測片収集 ・周回

すべてが物語表現です。

------------------------------------------------------------------------

# 2. プレイヤー

主人公は18人います。

しかし本当の主人公は

**プレイヤー自身**です。

プレイヤーは

18人の英雄の中から

「誰の答えで世界へ挑むか」

を選びます。

ゲーム開始メッセージ例

> 「誰の答えで、この世界へ挑みますか？」

この問いがゲーム全体のテーマになります。

------------------------------------------------------------------------

# 3. 世界

世界名：ミスラ

勢力

-   ライオネル
-   ソンガイ
-   ヴァナー
-   マグマー
-   アビサル
-   ヴェツルヴィアン

それぞれ価値観が異なります。

争いもあります。

しかし、それぞれが世界に必要な存在です。

------------------------------------------------------------------------

# 4. アルカナ

正式名称

終焉の観測者 アルカナ

性別

女性

## 正体

神ではありません。

魔王でもありません。

世界を記録する「記録官」です。

何度も滅びと再生を見続け、 その記憶だけを失わない存在です。

彼女だけが全ての歴史を覚えています。

忘れられないことが彼女の呪いです。

## 思想

「人は最後には諦める」

だから

世界を壊すのではなく

苦しみごと終わらせたい。

彼女は世界を憎んでいません。

世界を愛し過ぎた存在です。

------------------------------------------------------------------------

# 5. 観測片

観測片はストーリーの中心です。

設定資料ではありません。

アルカナの記憶そのものです。

各章で入手します。

最初は誰の記憶か分かりません。

1〜3章 女性の声だけ

4〜8章 後ろ姿

9〜11章 アルカナという名前

12章 姿が判明

20章 全て繋がる

ホーム画面には

「観測ライブラリ」

を実装し、

観測片を一覧閲覧できるようにします。

------------------------------------------------------------------------

# 7. 主人公18人

主人公は18人ですが、

18個の物語を作るのではありません。

18通りの「答え」を作ります。

例

アルディン 守る

カガチ 前へ進む

ヴェスナ 居場所

キャシヴァ 人生を楽しむ

サージ 未来は変えられる

全員の共通回答は

「それでも生きる」

です。

------------------------------------------------------------------------

# 8. ボス

ボスは悪役ではありません。

全員

アルカナと同じように

途中で諦めた存在です。

仲間になっても思想は変わりません。

主人公を認めるだけです。

つまり

改心ではなく

共闘です。

------------------------------------------------------------------------

# 9. 主人公差分

チャプターはどの順番でも遊べます。

主人公も自由に変更できます。

そのため

過去チャプター前提の会話は禁止。

主人公は

そのチャプターで得られる情報だけを知っています。

違いは

「反応」

です。

同じ観測片でも

主人公ごとに受け止め方が変わります。

------------------------------------------------------------------------

# 10. 特殊イベント

全主人公に

2つ程度の専用イベントを作ります。

例

アルディン Chapter1

カガチ Chapter8

イレーナ Chapter12

全主人公 Chapter20

主人公を変えて遊ぶ価値を作ります。

------------------------------------------------------------------------

# 11. ラスボス

20章では

観測片が全て共鳴します。

今まで仲間になったボス達が

アルカナへ

それぞれの答えを伝えます。

プレイヤーは戦いながら

アルカナの人生を知ります。

------------------------------------------------------------------------

# 12. エンディング

アルカナは最後まで

自分が完全に間違っていたとは思いません。

しかし

主人公達と

仲間になったボス達を見て

初めて迷います。

最後の台詞例

> 「続きを......観てみたいですね。」

クリア後

アルカナは最後の仲間として解放されます。

ボスは倒すと仲間になる。

そのルールは最後まで例外を作りません。

------------------------------------------------------------------------

# 13. 開発ルール

-   ゲームシステムで物語を語る
-   観測片は最重要ギミック
-   主人公18人は18通りの答え
-   ボスは改心しない
-   仲間化は主人公を認めた結果
-   アルカナは悪役ではない
-   プレイヤー自身も物語の登場人物である
-   周回するほど物語の全体像が見える設計にする
