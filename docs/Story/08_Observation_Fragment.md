## 文体変化

観測片は、章が進むほど「客観的な記録」から「アルカナ本人の記憶」へ近づきます。

| 範囲 | 文体 | 目的 |
| --- | --- | --- |
| Chapter1-3 | 無名の記録。固有名は最小限。 | まだ誰の記憶か分からない状態を保つ。 |
| Chapter4-8 | 感情が少し漏れる。白い髪、鐘、手紙などの反復語を置く。 | アルカナが単なる黒幕ではないと匂わせる。 |
| Chapter9-12 | 記録官、未来への手紙、観測者という語を出す。 | アルカナの正体へ橋をかける。 |
| Chapter13-18 | 記録が機械的処理に変質した怖さを出す。 | 白紙戦争とメカゾ0ア連章を接続する。 |
| Chapter19-20 | ネリ、オルハ、白紙戦争が明確につながる。 | アルカナの個人的な痛みとして回収する。 |

個人的な喪失の正本は `12_Arcana_Private_History.md` を参照。

## 全文

### Fragment 01: 白門の慈悲

```text
終焉は慈悲だと、誰かが言った。
ネリは鐘楼の下で、弟の笑い方まで書いてほしいと言った。
倒れた子の名を呼ぶ声も、瓦礫の下で途切れた祈りも、もう増えなくて済むから。
けれど私は知っている。
増えない痛みは、癒える時間も持たない。
鐘は鳴らなかった。
それでも、誰かが鐘楼へ登ろうとしていた。
```

### Fragment 02: 止まる門

```text
進めば、また傷つく。
止まれば、もう失わない。
石はそう言って、道の真ん中で眠っていた。
私は一度、その静けさを美しいと思った。
だが石の影に座った者は、助けを呼ぶ声にも届かなかった。
```

### Fragment 03: 祈りの名

```text
あの方は、終わりを罰として語らなかった。
あの方は、眠りを救いとして語った。
だから多くの者が膝をついた。
自分で選ばなくていい祈りは、とても軽かった。
軽すぎて、立ち上がる足の重さを忘れてしまった。
```

### Fragment 04: 眠らない熱

```text
獣は眠りの匂いを嫌った。
火山の腹で、心臓のような音が鳴っていた。
生き物は、痛みを避けるためだけに生まれたのではない。
腹が減り、牙が伸び、誰かの名を知らずに吠える。
それでも熱は、終わりより先に朝を探した。
```

### Fragment 05: 角笛

```text
争いは世界を傷つける。
それは何度観ても変わらない。
けれど戦場で、誰かは誰かのために立った。
剣戟の奥に、命を確かめる笑いがあった。
私はその音を、消すべき騒音と呼べなかった。
```

### Fragment 06: 変われない群れ

```text
群れは同じ方向へ走る。
一頭が止まれば、波に飲まれる。
それでも母は、遅れた子を振り返った。
世界は変われないのか。
その問いに答える前に、子は一歩だけ違う足跡をつけた。
```

### Fragment 07: 別れの手

```text
別れは必ず来る。
だから手を伸ばさない方が、賢いのかもしれない。
だが空の手は、失わない代わりに何も覚えない。
握った温度は消える。
それでも温度があったことは、世界のどこかに残る。
```

### Fragment 08: 生の証

```text
痛い。
腹が減る。
転んで、泥がつき、笑われる。
私はそれを惨めだと思った。
けれど誰かが言った。
それ、全部生きてる証よ。
その言葉は呪いより強く、眠りより騒がしかった。
```

### Fragment 09: 優しい眠り

```text
休んでいい、と声がした。
それは命令ではなく、祈りのように聞こえた。
傷ついた者ほど、その声に目を閉じた。
私はその優しさを責められなかった。
責められないからこそ、誰が目覚めを守るのかを考えた。
```

### Fragment 10: 誤差

```text
誤差を消せば、悲劇は減る。
記録はそう証明した。
だが誤差の中に、誰かの笑い癖があった。
予定外の寄り道が、命を救った日があった。
未確定要素は危険だ。
そして、未来でもあった。
```

### Fragment 11: 愛したから

```text
彼女は世界を憎んでいなかった。
憎んでいたなら、もっと簡単だった。
オルハは、記録は未来への手紙だと言った。
愛して、記録して、また失って、なお見続けた。
砂に残る足跡が消えるたび、彼女は新しいページを開いた。
いつしかページを閉じることが、愛に見えてしまった。
```

### Fragment 12: 観測者

```text
終焉の観測者。
その名は、壊す者の名ではない。
見届ける者の名だった。
だが観測は中立ではいられない。
同じ悲劇を何度も視た目は、いつか結末を書き換えたくなる。
あるいは、結末だけを先に閉じたくなる。
```

### Fragment 13: 翼の規格

```text
自由は危険だ。
飛ぶ者は落ち、落ちた者は誰かを巻き込む。
ならば翼の角度を固定すればいい。
風の強さも、行き先も、許可された範囲に収めればいい。
だが固定された翼は、もう空を選べなかった。
```

### Fragment 14: 切断

```text
迷いを切る。
後悔を切る。
痛みを切る。
刃はいつも、早い答えを持っていた。
けれど切り落とされたものの中に、次へ進む理由も混じっていた。
効率は、時々とても静かな暴力になる。
```

### Fragment 15: 完全体

```text
欠けたものを全て埋めれば、完全になる。
完全なら、もう苦しまない。
機械神はそう証明しようとした。
だが完全な沈黙には、誰の名前もなかった。
私はその完成を、救いと呼べなかった。
```

### Fragment 16: 命令だけが残る

```text
神は倒れた。
けれど命令は残った。
考えなくていい、従えばいい、痛みは処理される。
壊れた兜は、持ち主のいない声でそう繰り返した。
思考のない安堵は、眠りとよく似ていた。
```

### Fragment 17: 同じ型

```text
名前を消せば、争いは減る。
違いを消せば、妬みも恐れも減る。
同じ形の兵が、同じ歩幅で進んだ。
だが誰も呼び返せなかった。
戻る場所には、名を呼ぶ声が必要だった。
```

### Fragment 18: 照準

```text
終わりは照準された。
苦しみを生む世界ごと撃ち抜けば、痛みは終わる。
その計算は正しいのかもしれない。
かつて私の記録は、砲台の角度を決めるために読まれた。
だが照準の先で、誰かがまだ朝食の匂いを覚えていた。
世界は、式より少しだけ騒がしかった。
```

### Fragment 19: 増える終わり

```text
終わりは一つでよかった。
閉じれば静かになるはずだった。
だが終わりは増えた。
首を伸ばし、声を重ね、止める者の手から滑り落ちた。
終幕さえ、制御できないものになった。
```

### Fragment 20: 余白

```text
全ての観測片が、彼女の記憶だった。
愛した世界、失った名前、閉じようとしたページ。
ネリの鐘。オルハの手紙。白紙に戻した戦争。
彼女は最後まで、自分が完全に間違っていたとは思わなかった。
ただ一つ、空白の行を見つけた。
そこにはまだ、続きを書ける余白があった。
```

- Fragment04-08: 白い髪、後ろ姿、記録官の気配だけを出す。
- Fragment09-11: アルカナの名を断片的に出す。
- Fragment12: `終焉の観測者 アルカナ` の表示を解禁。
- Fragment20: 20個の断片が一つの記憶として並び替わる。
