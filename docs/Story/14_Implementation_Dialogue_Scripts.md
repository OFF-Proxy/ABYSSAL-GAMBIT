```text
[interlude=INT_20_ARCANA_FINAL]
（ト書き: 全ての観測片が宙に開く。白いページの奥で、古い鐘の音が一度だけ鳴る）
アルカナ: まだ歩くのですね。すべてを観たうえで、あなたは何を選びますか。
主人公: 終わりではない答えを。
アルカナ: 私は、ネリという子の名をまだ覚えています。
アルカナ: 世界は忘れました。村も、鐘も、弟の笑い方も。
アルディン: 覚えているだけでは守れない。だが、忘れたら守る理由も消える。
アルカナ: オルハは、記録は未来への手紙だと言いました。
アルカナ: けれど手紙は、何度も燃やされました。
ジラーン: 燃え残った言葉で、次の手を温めることもできます。
アルカナ: 私の記録は、次の戦争を避けるためにありました。
アルカナ: けれど人は、それで次の戦争を上手に始めました。
イレーナ: 記録は免罪符ではありません。けれど、諦めの証拠だけでもありません。
カガチ: 上手に傷つける奴がいるなら、下手でも止める奴が要る。
レヴァ: 未来が危ないから固定する？ それ、空を檻にするのと同じだよ。
シダイ: 見えない影にも名がある。記録に残らなくても、支えた事実は消えない。
ヴェスナ: 帰る場所は、最初から残っているものじゃない。誰かと火を置いて作るものだ。
カーラ: 戻るまで待つ時間も必要だ。眠らせれば、その時間まで奪う。
ブローム: 違う者同士でも、同じ明日を支えられる。完全でなくても、軍列は進む。
ヴァース: 腹が減る。血が熱い。なら、まだ終わりじゃねえ。
スターホーン: 退屈な終わりより、騒がしい明日だ。角を鳴らせ。
ラグノラ: 群れは飲み込む腹ではない。帰る場所として、子を待つ。
リリス: 別れが来るからこそ、手を伸ばした事実は残るわ。
キャシヴァ: ほらね。痛くて、空腹で、笑えるくらい格好悪い。だから生きてるのよ。
メーヴ: 休ませたい気持ちは否定しません。けれど、選ぶ権利は残します。
ジリックス: 未確定要素、多数。排除ではなく、保留を推奨します。
サージ: 砂に消える足跡でも、歩いた熱までは消えません。
サイオン: 観測された結末でも、足運びは選べる。舞はまだ終わりません。
アルカナ: どうして、そこまで。
アルカナ: あなたたちは、また失うかもしれないのに。
（主人公差し替え: 選択中主人公の最後の答え）
アルカナ: ……そうですか。
（ト書き: 終焉の書の白紙に、一行だけ余白が残る）
アルカナ: ネリ。オルハ。白紙にした戦争。
アルカナ: どれも、終わらせれば増えない傷でした。
主人公: 増えない傷は、癒える時間も持たない。
アルカナ: ……それは、私が最初に書いた言葉でしたね。
アルカナ: では、もう一度だけ。
アルカナ: あなたたちの続きを、観てみましょう。
```

## 実装上の注意

- ここにある台詞は日本語正本。
- 英語化する場合も、アルカナは命令口調にしない。
- メカゾ0アは機械的でよいが、章ごとの担当主人公だけは感情の温度を上げる。
- Chapter20のフル台本は長いため、戦闘後幕間または複数ページVNで表示する。
- 既存 `INT_13` を置き換える場合、旧台詞は資料として残し、実装参照先だけをこの正本へ寄せる。
