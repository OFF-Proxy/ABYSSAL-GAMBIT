# DESIGN_R3-bossfeel: 章ボス固有ギミック／ヒーロー必殺／仲間化ボス育成パッシブ

> 設計＋実装: Claude (Cowork) / 2026-06-03（おすすめ提案 1・2・3 の実装）
> 状態: ✅ 実装済み（コンパイル0）。残: 実機調整・バランス(R3)・ボス別の個別化。

## 1. 章ボスの固有ギミック（予兆AoE）

- 章ボス戦（`IsBossWave`）開始時に `BossMechanicRoutine` を起動。一定間隔で味方を狙う「予兆付き範囲攻撃」を落とす。
- `BossAoeStrike`：味方の誰かの位置を中心に `PlayAreaIndicator` で予兆→1秒後に着弾。半径1.6マス内の味方に `TakeDamage`（110+章*60）＋0.6sスタン。
- 回数は章でスケール（2+章）。プレイヤーは散開／鉄壁陣形などで対応する“攻略要素”に。コア戦は対象外。
- 今後：ボスごとに固有のギミック（召喚・激昂・特殊弾）へ差し替えていく土台。

## 2. ヒーロー（主人公）＋1戦1回の必殺

- `heroUnitId`（既定 "Wolfpunch"）を毎ラン初手に確定付与（`GrantHeroUnit`、見つからなければランダムにフォールバック）。章スケール分はランダム追加。
- ヒーロー必殺「コマンドラリー」：`UseHeroUltimate()`＝盤面の味方全員へ シールド20%HP＋与ダメ+25%＋攻撃速度×1.2（6秒）＋発動エフェクト。**1戦に1回**（`heroUltUsedThisWave`、戦闘開始でリセット）。
- HUD：新規 `HeroUltButtonUI`（右下、戦闘中のみ表示。使用可で明るく/使用後はグレー）。`GameManager.IsHeroUltVisible`/`CanUseHeroUltimate()` を毎フレーム参照。
- オートバトラーに“能動的な操作”を1つ足す狙い。コア戦は当面非表示。

## 3. 仲間化ボスの育成パッシブ（前面化）

- 戦闘開始時 `ApplyRecruitedBossPassives()`：盤上の解放済みボス（`HasBossAlly`）にアフィニティ節目の固有パッシブを付与。
  - Lv3：開幕シールド（最大HP15%）
  - Lv5：攻撃速度 ×1.12
  - Lv8：与ダメ +12%
- 図鑑詳細の育成パネルに節目（●達成/○未達）を表示し、「倒す→育てる→強い固有効果」を可視化。
- 従来の一律ステータス倍率（`GetBossAffinityStatMultiplier`）に加えて、育成の手応えを節目で実感できるようにする。
